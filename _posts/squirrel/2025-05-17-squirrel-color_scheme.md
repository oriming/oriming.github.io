# Welcome

本文主要做笔记，记录鼠须管好看的主题。

演示环境:

+ macOS Sequoia 15.5
+ Squirrel 1.0.3

## 1. 配置主题

1. 打开配置文件
![配置文件地址](/assets/img/2025-05-17-squirrel/2025-05-17-07-59-11.png)

2. 找到主题标签 `style.color_scheme` 和 `style.color_scheme_dark`，并设定主题风格

```yaml
style:
  # 选择皮肤，亮色与暗色主题
  # color_scheme: onctf
  color_scheme: wechat
  color_scheme_dark: wechat_dark
```
